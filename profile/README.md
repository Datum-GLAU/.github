## 🌌 Who We Are

**Datum** is the **official data & technology club** of **GLA University, Mathura** — a student-powered open-source community where curiosity meets code. We are a collective of passionate developers, data scientists, ML engineers, and designers who build, break, and ship real-world projects together.

We believe in **learning by doing** — every project we tackle is open-source, collaborative, and designed to create actual impact.

> *"Data is the new oil, but collaboration is the engine."*

📊  Democratize data science education for students

## 🤝 Contributing

**Datum is community-first.** Whether you're a first-year writing your first `for` loop or a final-year deploying models to production — there's a place for you here.

### How to Get Involved

```bash
# 1. Explore our repositories
https://github.com/Datum-GLAU

# 2. Pick an issue labeled "good first issue" or "help wanted"

# 3. Fork → Branch → Code → PR

# 4. Join the discussion on our community channels
```

### Contribution Guidelines

- 📋 Read the `CONTRIBUTING.md` in each repo before starting
- 🌿 Always create a feature branch — never push directly to `main`
- 💬 Comment on an issue before working on it to avoid duplication
- 🧪 Write tests. Document your changes. Respect the code style
- 🎉 All PRs, big or small, are celebrated here
